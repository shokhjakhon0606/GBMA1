clistudy

clistudy is a simple command-line tool that helps you track how many minutes you actually study each day and each week.

Features

  • Log study sessions with one command
  • See today's total time per topic
  • See last 7 days summary per topic
  • No accounts, everything stored locally as JSON

Usage

$ clistudy log 60 "Rust exam prep"
Logged 60 minutes for 'Rust exam prep'.

$ clistudy today
Today's study summary:
- Rust exam prep: 60 min
-------------------------
Total: 60 min

$ clistudy week
Last 7 days summary:
- Rust exam prep: 240 min
-------------------------
Total: 240 min

Download

Download the binaries from the GitHub Actions artifacts or releases for your platform (Linux x86_64, Linux aarch64, etc.).

Pricing

clistudy is free and open source. In a real-world scenario, advanced features (CSV export, integrations, team reports) could be offered as a small paid upgrade, while the core tool stays free for students.
